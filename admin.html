How to use this Admin Panel

Step 1 — First time setup
Go to ⚙ Settings and set your Admin Password, Cloudinary config, and GitHub credentials.


Step 2 — Edit questions
Click any subject in the sidebar → find your question → click Edit. You can change text, options, correct answer, images, and explanation.


Step 3 — Upload images
In the edit modal, click the image upload area. Images go directly to Cloudinary and the URL is saved automatically in qi (question image) or ei (explanation image).


Step 4 — Save to GitHub
After saving a question, click Push to GitHub. This writes the updated JSON directly to your repo. Since your site fetches data/*.json from your repo, visitors see the change within seconds — no redeploy needed.


Step 5 — Add this file to your site
Place admin.html in your website root alongside index.html. Access it at yoursite.com/admin.html. The password protects it.


Your JSON file structure
Each subject is a separate file in data/ e.g. data/anatomy.json. The manifest at data/manifest.json tracks question counts per subject.
